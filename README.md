## Quick Start

1. Clone the repository:
    ```bash
    git clone https://github.com/wsmilian/internet-monitor.git
    cd internet-monitor
    ```

2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3. To run the connectivity monitor:
    ```bash
    python monitor.py
    ```
4. To run the speed test:
   ```bash
   python testspeed.py
    ```
